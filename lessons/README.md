# Syllabus

This folder holds the written lessons for the CS Academy Bioinformatics Project.

0. [**DNA as Data**](000_DNA_as_Data.md) — learn about how DNA is stored in the computer and how to calculate GC content.
1. [**k-mers**](001_kmers.md) — learn about finding genomic signatures using k-mers.
2. [**Sequence Alignment**](002_Sequence_Alignment.md) — an introduction to comparing DNA sequences pairwise.
3. [**Global Sequence Alignment Algorithm**](003_Global_Sequence_Alignment_Algorithm.md) — more in depth explanation of comparing sequence pairwise using the Global Sequence Alignment Algorithm.
